# StuntGuard Frontend

React + Vite + TypeScript frontend untuk dashboard skrining awal stunting.

## Setup

```bash
cd frontend
npm install
```

Opsional: buat `.env` untuk mengganti URL backend.

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Run

```bash
npm run dev
```

Frontend tersedia di `http://localhost:5173`.

## Build

```bash
npm run build
```
